# 📚 Book Store Web App

A modern, responsive **book listing web application** built with **Next.js (App Router)**, **React Context API**, and **Tailwind CSS**.

Users can:
✔ Browse books  
✔ Search by title or author  
✔ Filter by category  
✔ Paginate listings  
✔ Add books to cart  
✔ View cart with item counts and total price  

This project is ideal for frontend internship portfolios and demonstrates practical use of React state, routing, and UI design.

## 🧱 Features

### 📌 Main Functionality
- **Home / Books Page**
  - Search books (title/author)
  - Category filter
  - Pagination (4 items per page)
  - Book cards with image, price, rating, and Add to Cart
- **Cart Page**
  - Shows current items in cart
  - Increase / decrease quantities
  - Cart item count updates in Navbar
  - Total price calculation

### 💡 UX Enhancements
- Clean, modern UI with Tailwind CSS
- Responsive design (mobile/tablet/desktop)
- Accessible attributes (aria labels)
- Navbar cart count indicator
- Dark mode (if implemented)
